# Trademarks

Velkir is a Kubernetes operator that deploys and manages Valkey. It is a
separate project with its own name, maintainers, and release cadence.
This page records how the Valkey name is used here and states the
project's non-affiliation and packaging-status commitments.

## Valkey

Valkey® is a registered trademark of LF Projects, LLC. Velkir uses the
Valkey name only in a descriptive, nominative sense — to identify the
software that Velkir operates — and claims no rights in the mark.

## Non-affiliation

> Velkir is an independent project. It is not affiliated with, endorsed by, or sponsored by the Valkey project, LF Projects, LLC, or the Linux Foundation.

The sentence above is this project's **canonical non-affiliation
disclaimer**. It is reused verbatim in `README.md`, in the `NOTICE`
file, in both Helm charts' `Chart.yaml` annotations, and in the
operator image's OCI labels — keep every copy byte-identical.

## No logo, no Valkey-branded domain

Velkir ships **no** Valkey logo, wordmark image, or other Valkey brand
asset, and registers **no** Valkey-branded domain name. The operator's
API group is `velkir.ioxie.dev` and every project identifier uses the
Velkir name.

## Packaging status

Velkir does not, and will not, request **"Official"** status on Artifact
Hub or any equivalent operator registry. Velkir is listed only as a
community project, so no registry badge can contradict the
non-affiliation statement above.
